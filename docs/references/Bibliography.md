# Bibliography (Selected References)

This repository may reference third-party publications. To reduce redistribution and copyright risk, the repository does not include copyrighted PDFs by default. Where possible, cite the canonical publisher page, DOI, or official archive link.

## Citation Style
- Prefer DOI links where available.
- When referencing diseases or clinical concepts, avoid making efficacy claims and use neutral language.
- ICD-11: cite codes where relevant for taxonomy context only (not for clinical use).
- SNOMED CT: do not redistribute SNOMED CT content; if needed, reference concept identifiers only, subject to SNOMED licensing terms.
